# Frontend Engineering Keywords Reference

Use this as a comprehensive reference when checking keyword coverage for frontend or
frontend-heavy full-stack resumes. Focus on gaps that matter for the candidate's target
seniority and role.

---

## Role Terms
- Frontend Engineer, Frontend Developer, UI Engineer
- Full Stack Engineer, Full Stack Developer (frontend-leaning)
- React Developer, Next.js Developer
- JavaScript Engineer, TypeScript Engineer
- Senior / Staff / Principal (level qualifiers)

## Core Languages & Runtimes
- TypeScript, JavaScript (ES6+, ES2022)
- HTML5, semantic HTML, HTML accessibility attributes
- CSS3, modern CSS (Grid, Flexbox, custom properties / CSS variables)

## Frameworks & Libraries
- React (core), React 18, Concurrent Mode, Suspense
- Next.js (App Router, Pages Router), Remix, Gatsby
- Vue.js, Nuxt.js
- Angular, AngularJS (older)
- Svelte, SvelteKit
- Astro (static/hybrid)

## State Management
- Redux, Redux Toolkit (RTK)
- Zustand
- Jotai, Recoil
- MobX
- React Context API
- React Query (TanStack Query), SWR (data fetching + caching)
- XState (state machines)

## Rendering Strategies
- Server-side rendering (SSR)
- Static site generation (SSG)
- Incremental Static Regeneration (ISR)
- Client-side rendering (CSR)
- SPA (Single Page Application)
- Hydration, selective hydration, partial hydration
- Edge rendering, edge functions
- SEO optimization (meta tags, structured data, Open Graph)

## Performance Engineering
- Core Web Vitals (LCP, FID/INP, CLS)
- Lighthouse, PageSpeed Insights
- Lazy loading, code splitting, dynamic imports
- Bundle optimization, tree shaking, dead code elimination
- Image optimization (WebP, AVIF, next/image)
- Critical rendering path, above-the-fold optimization
- Prefetching, preloading, resource hints
- Service workers, Progressive Web App (PWA), offline support
- Caching strategy, CDN, edge caching
- Performance profiling, Chrome DevTools, Web Vitals

## Styling & Design Systems
- Tailwind CSS, utility-first CSS
- Material UI (MUI), Ant Design, Chakra UI, shadcn/ui
- CSS Modules
- Styled-components, Emotion (CSS-in-JS)
- SCSS / Sass
- Design tokens, theming, dark mode
- Responsive design, mobile-first design
- Design system, component library

## Accessibility (a11y)
- WCAG 2.1 / WCAG 2.2 (AA, AAA)
- ARIA attributes, ARIA roles, ARIA live regions
- Semantic HTML, landmark regions
- Keyboard navigation, focus management
- Screen reader testing (NVDA, JAWS, VoiceOver)
- Color contrast, visual accessibility
- Axe, Lighthouse accessibility audit

## Tooling & Build Systems
- Vite, Webpack, Turbopack, esbuild, Rollup
- Babel, SWC (transpilers)
- ESLint, Prettier (code quality)
- TypeScript compiler, tsconfig
- npm, yarn, pnpm, Bun (package managers)
- Monorepo: Turborepo, Nx, Lerna
- pnpm workspaces, npm workspaces

## Testing
- Jest, Vitest (unit testing)
- React Testing Library (RTL)
- Cypress, Playwright (E2E testing)
- Storybook (component testing, visual testing)
- Chromatic, Percy (visual regression testing)
- Mock Service Worker (MSW) (API mocking)
- Test-driven development (TDD)
- Code coverage

## APIs & Data Fetching
- REST API, RESTful API
- GraphQL, Apollo Client, urql, Relay
- Axios, fetch API, XMLHttpRequest
- WebSockets, Server-Sent Events (SSE)
- React Query / TanStack Query, SWR
- OpenAPI / Swagger (schema-driven API integration)

## UI / UX Collaboration
- Figma, Adobe XD, Sketch (design tool integration)
- Storybook (component documentation)
- Zeplin (design handoff)
- Prototyping, wireframing
- User research, usability testing (if involved)
- Design system contributions

## Mobile & Cross-Platform (adjacent skills)
- React Native (mobile apps from React knowledge)
- Expo
- Responsive design vs. adaptive design
- Progressive Web App (PWA)

## Animation & Interaction
- Framer Motion, React Spring (animation libraries)
- CSS animations, CSS transitions
- GSAP
- Micro-interactions, UX animation

## DevOps & Deployment (frontend-relevant)
- Vercel, Netlify, Cloudflare Pages (frontend hosting)
- CI/CD pipelines (GitHub Actions, Vercel, Netlify deploys)
- Docker (containerization for frontend preview envs)
- Environment variables, feature flags
- A/B testing, experimentation
- Web analytics (Google Analytics, Mixpanel, Amplitude)

## Browser & Web Platform
- Browser compatibility, cross-browser testing
- BrowserStack, Sauce Labs
- Browser storage: localStorage, sessionStorage, IndexedDB, cookies
- Web Workers, SharedArrayBuffer
- Web APIs: Intersection Observer, Resize Observer, MutationObserver
- Chrome DevTools, Firefox DevTools

## Collaboration & Process
- Git, GitHub, GitLab, Bitbucket
- Agile, Scrum, sprint planning
- Jira, Linear, Notion (project tracking)
- Code review, pull requests
- Figma collaboration, design-to-code handoff
- Pair programming, tech talks, knowledge sharing
