# software-engineer-ats-resume-optimizer

ATS resume analyzer for software engineers — scores your resume, surfaces keyword gaps, and rewrites weak bullets.

A Claude Code skill that analyzes software engineering resumes for ATS (Applicant Tracking System) compatibility and gives you a clear action plan to improve your chances of getting through to a human recruiter.

Works for backend, frontend, and full-stack engineers. Paste your resume — with or without a job description — and get back:

- **ATS score** (0–100%) with a section-by-section breakdown
- **Keyword gap analysis** — what's missing vs. what top JDs expect
- **Rewritten bullet points** you can copy-paste directly
- **Top 5–10 prioritized fixes** to reach ≥90% ATS compatibility

Covers roles at fintech companies, startups, and big tech. Knows the keyword landscape for Node.js, TypeScript, NestJS, React, Next.js, AWS, and the broader backend/frontend ecosystem.

## Install

```bash
npx skills add solomon-igbigbi/software-engineer-ats-resume-optimizer
```

## Usage

Just paste your resume into Claude Code and ask anything — the skill triggers automatically:

> *"Can you review my resume? I'm applying to senior backend roles."*
>
> *"What keywords am I missing for this job description?"*
>
> *"Can you rewrite my bullet points? They feel weak."*
>
> *"Score my resume out of 100."*
